## [1.0.0] - 2026-01-30

### 首次发布

这是一个完整的 RPA 机器人流程自动化管理平台。

#### 新增功能

- **账号管理**
  - 管理多个影刀机器人账号
  - 账号状态追踪（pending/running/completed/failed）
  - 账号任务统计

- **任务控制**
  - 创建、编辑、启动、停止 RPA 任务
  - 任务配置管理
  - 任务状态实时更新

- **执行日志**
  - 查看任务执行记录
  - 截图和日志详情展示
  - 执行时长统计

- **仪表盘**
  - 实时统计数据展示
  - 性能趋势图表
  - 成功率统计

- **实时通信**
  - Server-Sent Events (SSE) 实时更新
  - 无需刷新页面，数据自动同步

- **云端存储**
  - 阿里云 OSS 集成
  - 截图和日志云端存储

#### 技术栈

- **前端**: React 18 + TypeScript + Vite + TailwindCSS
- **后端**: FastAPI + Python 3.10+ + SQLAlchemy (异步)
- **数据库**: SQLite (aiosqlite)
- **存储**: 阿里云 OSS
