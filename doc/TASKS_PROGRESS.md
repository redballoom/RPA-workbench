### 8.4 工作流程

```
用户点击启动
    │
    ▼
调用 /resources/proxy/intranet (后端转发到影刀)
    │
    ▼
【不更新状态】等待影刀调用 /webhook/confirm
    │
    ▼
/webhook/confirm 调用 → 状态变为 running
    │
    ▼
执行完成 → /webhook/execution-complete → 状态变为 pending
```
